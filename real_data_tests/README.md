# Real Data Tests — P12

Test script: `test_p12_real_data.py`

All tests use **real data downloaded from public internet sources** (NOAA, PhysioNet, SNAP, GBIF, NIST, FCC, Ofcom, etc.).
No synthetic data is used in any test.

## Running
```bash
python test_p12_real_data.py
```

Results are saved to `figures_p12/`
